# Toolbox
A utility inspired config for MacOS. Ideal for Macbooks with a notch.

Must install Aušrinė font from: https://berzulis.com/ or change items/spaces.sh font to "SF Pro:SemiBold:12"

## Tools Used

- Alacritty
- Fish Shell
- Aerospace
- Sketchybar
- Neofetch
- Raycast
- Jankyborders
- Spicetify
- Thunderbird
